How Can We Help You?

About This Page

This help page is created to assist users in understanding how to use the Urban Safety Drive website effectively. If you have any confusion, doubts, or need guidance, this section will provide all necessary information. Our aim is to make your experience smooth and informative.

How to Use This Website

  • Use the navigation bar to move between pages.
  • Visit the Rules page to learn safety guidelines.
  • Visit Causes page to understand accident reasons.
  • Use Contact page to send your queries.
  • Check Help page for guidance.

Governments Helpline numbers

  • National highway Helpline No: 1033
  • Emergancy Response service: 108
  • For Ambulance Service : 102
